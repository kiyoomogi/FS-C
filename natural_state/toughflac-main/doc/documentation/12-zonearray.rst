.. _zonearray-doc:

Zonearray
=========

Submodule :mod:`zonearray` provides functions to easily access extra variable arrays saved during the coupling and is organized as follow:

.. code-block:: text

    zonearray
     ├─permeability
     ├─porosity
     ├─porosity_initial
     ├─porosity_delta
     ├─pp
     ├─pp_equivalent
     ├─pp_delta
     ├─temperature
     ├─temperature_delta
     ├─strain_vol
     ├─stress_delta
     ├─stress_delta_flat
     ├─stress_delta_prin_x
     ├─stress_delta_prin_y
     ├─stress_delta_prin_z
     ├─stress_delta_min
     ├─stress_delta_max
     ├─stress_delta_mean
     ├─density
     ├─biot
     ├─therm_coeff
     ├─saturation
     └─pcap
